Step 1. Install Python

Step 2. Set up a virtual environment
          "cd EV_Dashboard"(Navigate to this folder)
          "python -m venv venv"

Step 3. Install the Libraries( in command prompt)
"pip install streamlit pandas altair plotly scikit-learn"

Step 4. To launch dashboard navigate to this folder(EV_Dashboard) location in command prompt.

Step 5. Then run the following command: 
           "streamlit run ev_dashboard.py"
