Electric Vehicle Adoption Tools (EVAT)

Accelerating Australia's EV Future through AI, ML & Smart Applications

About the Project

The EV Adoption Tools project is dedicated to promoting the adoption of Electric Vehicles (EVs) in Australia. By building smart tools and leveraging data science, the EVAT team aims to enhance user experience, improve infrastructure planning, and make EV usage more practical and appealing.

Trimester T1 2025 Goals

  • Develop App MVP with intuitive navigation & filters
  • Predict charger availability using AI (ARIMA, LSTM, XGBoost)
  • Identify blackspots using clustering models (DBSCAN, KMeans)
  • Enable real-time infrastructure monitoring with Power BI
  • Build AI-powered chatbot for smart recommendations

Deliverables Overview

  • Interactive charger locator & booking system
  • Forecasting APIs deployed with Flask & MongoDB
  • Interactive dashboards with real-time insights
  • NLP-based AI Chatbot integrated with app
